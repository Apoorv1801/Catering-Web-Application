# 🍽️ Royal Caterers – Catering Web App

## 📌 Overview

A full-stack catering web application that allows users to browse menus, add items to cart, and book catering services. Includes an admin panel for managing menu items and bookings.

---

## 🚀 Features

* Menu browsing with categories
* Cart & booking system
* User login/signup
* Admin dashboard (add/manage food items)
* Responsive design

---

## 🧑‍💻 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Java, Spring Boot
* Database: MySQL

---

## ⚙️ Setup

1. Configure MySQL in `application.properties`
2. Run Spring Boot backend (`localhost:8080`)
3. Open frontend with Live Server
